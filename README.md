# Stellar Burgers
## SPA-приложение космической бургерной

## Стек
- TypeScript
- React
- Redux(Redux Toolkit)

## Тесты
- Jest tests
- Cypress tests

### Особенности 
- Регистрация / Авторизация
- Восстановление / Сброс пароля
- Лента заказов (общая / индивидуальная)
- DnD при сборке бургера

### Ссылки
- [Ссылка на проект](https://enrja.github.io/react-burger/)

### Запук проекта

#### `npm install`

#### `npm start`
